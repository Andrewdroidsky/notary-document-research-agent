Фильтр 3. Убираются общие фоновые, дублирующие и несобственно-профильные акты.
Исключаются из итогового корпуса:
акты по биометрической идентификации;
общие акты ПОД/ФТ;
общие акты о персональных данных и информационной безопасности;
материалы ЕГРН и Росреестра, не образующие специальный порядок для копий;
общие паспортные и миграционные регламенты без собственного специального правила по данной подтеме.
