1. Росфинмониторинг.
Статус: НЕПРИМЕНИМО.
Для базовой подтемы о свидетельствовании верности копий отдельный обязательный слой Росфинмониторинга не формирует специального порядка совершения именно этого нотариального действия.

2. Банк России.
Статус: НЕПРИМЕНИМО.
Банковский регулятор не формирует специальный порядок свидетельствования верности копий документов и выписок из них.

4. Акты нотариальных палат субъектов.
Статус: НЕ ВЫЯВЛЕНО без указания конкретного субъекта Российской Федерации.
Для данной подтемы региональный уровень может давать организационные и тарифные материалы, но без субъекта РФ его нельзя корректно индивидуализировать.
