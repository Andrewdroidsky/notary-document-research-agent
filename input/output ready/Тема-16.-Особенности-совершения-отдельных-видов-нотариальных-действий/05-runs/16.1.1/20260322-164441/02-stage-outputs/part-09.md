Дельта-аудит после fresh external search.

1. Новых подтвержденных документов сверх корпуса, уже собранного в Частях 2-8, не выявлено.
Это означает, что после повторного внешнего прохода не найден новый самостоятельный федеральный, ведомственный или международный акт, который менял бы структуру базового корпуса.

2. Подтверждено, что цепочка актуализации приказов Минюста по данной подтеме состоит из базовых приказов N 156 и N 226 и их свежих поправок N 12, N 13, N 204 и N 205.
```text
12:
https://rg.ru/documents/2025/02/03/minjust-prikaz12-site-dok.html

13:
https://rg.ru/documents/2025/02/03/minjust-prikaz13-site-dok.html

204:
https://rg.ru/documents/2025/08/31/minjust-prikaz204-site-dok.html

205:
https://rg.ru/documents/2025/08/31/minjust-prikaz205-site-dok.html
```

3. Подтверждено отсутствие необходимости возвращать в основной корпус акты о биометрической идентификации, общие акты ПОД/ФТ, материалы ЕГРН и иные общие фоновые документы.
Причина: они не дают специального правила именно для свидетельствования верности копий документов, копий с копий документов и выписок из них.

4. Специальный отдельный акт Пленума Верховного Суда РФ по подтеме по итогам повторного fresh pass также не выявлен.
