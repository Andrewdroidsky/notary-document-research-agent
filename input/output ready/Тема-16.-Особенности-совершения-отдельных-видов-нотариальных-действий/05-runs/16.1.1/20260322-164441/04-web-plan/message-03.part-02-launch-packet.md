Все элементы ссылочного формата — только внутри code-блоков.

– упоминаний доменов, имён сайтов или путей вне CODE-БЛОКОВ, независимо от наличия "http"/"www".

3\. Обнаружение любого элемента ссылки вне CODE-БЛОКА делает ответ НЕДЕЙСТВИТЕЛЬНЫМ. Ответ обязан быть ПЕРЕПИСАН до полного соответствия.

**В строках VERIFIED/Заголовок/Сверка запрещено писать URL; URL1 и URL2 указывать только отдельными строками внутри code-блоков.**

**ЖЁСТКОЕ УСЛОВИЕ ВЫХОДА (BLOCKER):**

**(а) “VERIFIED URL2: ДА/НЕТ”,**

**(б) “Заголовок страницы URL2 (как на странице)”,**

**(в) “Сверка реквизитов: дата/номер/полное название — совпало/не совпало”.**

**Если VERIFIED URL2 ≠ ДА — URL2 не выдаётся, ставится “URL2: отсутствует (КАРАНТИН)” + каноническая строка поиска с обязательным указанием структурного элемента (глава/статья/пункт/подпункт/приложение).**

## Внутренние guardrails этой подтемы

- Тип подтемы: `substantive`
- Фокус: `Свидетельствование верности копий документов, копий с копий документов и выписок из них`
- Правило: Самостоятельный тарифный разбор не смешивается с базовой подтемой, если для группы есть отдельный тарифный подпункт.
- Не смешивать самостоятельный тарифный разбор с базовой подтемой: тариф вынесен в `16.1.2`.
