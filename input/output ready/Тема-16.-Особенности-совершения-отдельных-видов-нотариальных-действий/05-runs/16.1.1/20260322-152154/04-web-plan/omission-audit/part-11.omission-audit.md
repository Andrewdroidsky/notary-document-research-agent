## Omission Audit Layer: Часть 11. ПЕРЕЧЕНЬ ПРАКТИЧЕСКИХ ЗАДАНИЙ ПО ЗАДАННОЙ ТЕМЕ

Это второй проход на пропуски. Его цель — не переписать предыдущие части, а найти то, что еще недожато или выглядит слабо подтвержденным.

Снимок текущего покрытия:
- Подтема: `16.1.1. Свидетельствование верности копий документов, копий с копий документов и выписок из них.`
- Тип подтемы: `substantive`
- Фокус: `Свидетельствование верности копий документов, копий с копий документов и выписок из них`
- Подтвержденные части: нет
- Часть 2: карточек документов = 0, карантин = False
- Часть 3: блоков закрыто = 0 из 37
- Часть 4: диапазонов закрыто = 0 из 4
- Часть 5: диапазонов закрыто = 0 из 3
- Часть 6 подтверждена = False
- Часть 7 подтверждена = False
- Часть 8 подтверждена = False

Подозрительные места и флаги:
- В Части 2 отсутствуют обязательные маркеры: АНАЛИЗ ОБЛАСТИ ПРАВА, АНАЛИЗ-КОДЕКСЫ И БАЗОВЫЕ АКТЫ, A. РЕГУЛЯТОРНОЕ ЯДРО, B. ОПОРНЫЕ ДОКУМЕНТЫ, КАРАНТИН, FAIL-SAFE CHECK
- Часть 4 закрыта не полностью: 0 из 4 диапазонов.
- Часть 5 закрыта не полностью: 0 из 3 диапазонов.
- Часть 6 пока не подтверждена; omission audit должен учитывать этот пробел.
- Часть 7 пока не подтверждена; omission audit должен учитывать этот пробел.
- Часть 8 пока не подтверждена; omission audit должен учитывать этот пробел.
- У подтемы есть отдельный тарифный sibling `16.1.2`: omission audit не должен смешивать тарифный добор с базовой подтемой.

Контрольные вопросы второго прохода:
- Какие обязательные смысловые узлы темы выглядят недожатыми даже после Частей 2–8?
- Какие обязательные блоки, органы или слои пройдены формально, но слабо доказаны документами?
- Где фильтры и дедуп могли убрать полезный документ слишком рано?
- Какие карантинные или слабо верифицированные позиции стоит перепроверить отдельным добором?
- Какие скрытые режимы темы могли остаться вне поля зрения: международный, цифровой, контрольный, процессуальный, финансовый?

Режим omission audit:
- Искать не количество документов, а реальный недожатый смысловой слой.
- Проверять не только отсутствие документа, но и слабость верификации уже найденного документа.
- Не повторять уже сильные карточки без новой дельты.
