# Reasoning-layer: Часть 7. ФИЛЬТР БЕЗ ПОВТОРЕНИЙ

Цель reasoning-слоя: Добавить только действительно новые документы без косметического размножения уже найденной базы.

Контекст подтемы:
- Подтема: `16.1.1. Свидетельствование верности копий документов, копий с копий документов и выписок из них.`
- Тип подтемы: `substantive`
- Фокус: `Свидетельствование верности копий документов, копий с копий документов и выписок из них`
- Подтвержденные предыдущие части: нет подтвержденных предыдущих частей
- Базовое правило исполнения: Самостоятельный тарифный разбор не смешивается с базовой подтемой, если для группы есть отдельный тарифный подпункт.
- Связанная тарифная подтема: `16.1.2. Исчисление размера федерального и регионального тарифа при свидетельствовании верности копий документов и выписок из них.`

Как думать на этой стадии:
- Считать новым только документ, который дает новый нормативный слой, новый орган, новый режим или новый структурный элемент.
- Не принимать за 'новый' тот же акт в другой редакции или на другой площадке без новой смысловой функции.
- Проверять, закрывает ли документ ранее незакрытый риск или узел темы.

Типичные пропуски и ошибки:
- Добавление псевдо-новых документов ради количества.
- Подмена новых документов новыми ссылками на старые акты.
- Потеря точечной, но ценной нормы из-за слишком грубого дедупа.

Общие запреты reasoning-слоя:
- Не придумывать документы, роли или связи без подтвержденной документной опоры.
- Не ослаблять охват только потому, что предыдущие части уже содержат большой объем материала.
- Не превращать reasoning-layer в мета-комментарий вместо исполнения текущей Части.
